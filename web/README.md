# Icarus magic — Web

Vue 3 + Vite + Less + Element Plus + Vue Router。

## 开发

```bash
cd web/Icarus-magic-web
npm install
npm run dev
```

## 构建

```bash
npm run build
npm run preview
```

## 结构说明

- `src/layouts/MainLayout.vue`：顶栏（平台名 **Icarus magic**）+ 主内容区
- `src/views/Home.vue`：默认首页占位，待后续开发
- `src/router/index.js`：路由配置
